# 故事坊 UI 优化 - 完整设计文档

## 设计原则回顾

### 老年人 UI 核心理念
```
温暖（Warm）—— 不是冷冰冰的科技，而是有人情味
引导（Guide）—— 不是让用户自己摸索，而是一步一步陪
惊喜（Delight）—— 不是"能用就行"，而是"哇，还可以这样"
```

### 认知心理学基础
- **菲茨定律**：点击目标要大（最小 48×48px，建议 60-80px）
- **希克定律**：选择越多，决策时间越长，最多 2-3 个可选项
- **认知负荷**：同一屏幕信息量控制在 3-5 个核心元素
- **习惯与安全感**：老年人依赖"已知模式"，害怕"弄坏东西"

### 视觉设计要求
- 字体最小 18px，标题 24-32px
- 行高 1.6-1.8，段落间距充足
- 对比度 ≥ 4.5:1（WCAG AA 标准）
- 暖色系为主（橙、黄、红）—— 感知更清晰，情绪更积极
- 图标 + 文字标签，永远不要只用图标

**JavaScript 实现**：
```javascript
// 页面切换动画
function switchPage(fromPage, toPage) {
    fromPage.classList.add('page--leaving');

    fromPage.addEventListener('animationend', () => {
        fromPage.classList.remove('page--leaving');
        fromPage.classList.add('hidden');

        toPage.classList.remove('hidden');
        toPage.classList.add('page--entering');

        toPage.addEventListener('animationend', () => {
            toPage.classList.remove('page--entering');
        }, { once: true });
    }, { once: true });
}
```

### 3.2 音效反馈

**目标**：操作时提供声音反馈

**音效文件**：
- `click.mp3` - 按钮点击
- `start-record.mp3` - 开始录音
- `stop-record.mp3` - 停止录音
- `success.mp3` - 操作成功
- `error.mp3` - 操作失败

**JavaScript 实现**：
```javascript
class SoundManager {
    constructor() {
        this.sounds = {};
        this.enabled = true;
    }

    load(name, url) {
        this.sounds[name] = new Audio(url);
        this.sounds[name].preload = 'auto';
    }

    play(name) {
        if (!this.enabled || !this.sounds[name]) return;

        // 克隆音频以支持快速连续播放
        const sound = this.sounds[name].cloneNode();
        sound.volume = 0.5;
        sound.play().catch(() => {
            // 忽略自动播放限制错误
        });
    }

    toggle() {
        this.enabled = !this.enabled;
        return this.enabled;
    }
}

// 使用示例
const sounds = new SoundManager();
sounds.load('click', '/sounds/click.mp3');
sounds.load('start-record', '/sounds/start-record.mp3');
sounds.load('stop-record', '/sounds/stop-record.mp3');
sounds.load('success', '/sounds/success.mp3');
sounds.load('error', '/sounds/error.mp3');

// 为按钮添加音效
document.querySelectorAll('button').forEach(btn => {
    btn.addEventListener('click', () => sounds.play('click'));
});

// 录音按钮特殊音效
micBtn.addEventListener('mousedown', () => sounds.play('start-record'));
micBtn.addEventListener('mouseup', () => sounds.play('stop-record'));
```
